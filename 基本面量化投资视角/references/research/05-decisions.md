# 决策框架：基本面量化投资策略

> 来源：《基本面量化投资策略》董鹏飞，中信出版集团 2022年
> 回测区间：1998年12月31日至2020年9月30日（~22年）
> 涵盖 53 个单因子指标、31 个多因子指标的实证结论

---

## 1. 投资的第一性原理（5个基本维度）

攻防兼备，用于实现"先为不可胜，以待敌之可胜"：

| 维度 | 核心原则 | 具体做法 |
|------|---------|---------|
| **维度1：能力圈内做投资** | 只在能力圈内重仓，远胜圈外乱买 | 能力圈是动态变化的，需持续扩充（参考巴菲特从格雷厄姆到费雪到芒格的演进） |
| **维度2：投资就是买企业** | 以股东心态投资（段永平：问自己是否愿意买下整个公司） | 研究基本面信息，剔除有瑕疵的企业，选择运营质量较高的企业 |
| **维度3：要留有安全边际** | "你支付的是价格，获取的是价值"（巴菲特） | 买入价应低于公允价值，吸收预测失误或运气不佳的影响 |
| **维度4：正确看待价格波动** | 市场先生情绪不稳定，但价格终向价值回归 | 短期波动不等于损失，只有在低谷减仓才导致真正损失 |
| **维度5：相信时间的力量** | 市场恢复理性一般需要2-3年（格林布拉特） | 长期看来市场会自动走向正确，但需确保短期内能幸免于难 |

**核心公式**：投资业绩 = f(投资策略, 决策执行, 运气)，长期来看运气的影响趋近于零。

## 2. 系统性投资流程：逐步构建方法论

### 2.1 自下而上的策略构建步骤

**第一步：理解投资第一目的**
- 投资就是为了赚钱（税后总回报最大化）
- 不要被阿尔法、贝塔、夏普比率等绩效指标主导——它们是衡量工具，不是目的本身

**第二步：建立第一性原理思维**
- 像芒格一样积累多元思维模型（格栅理论）
- 把多个维度的模型综合使用，注意"合奏效应"（lollapalooza）

**第三步：从单因子指标开始认知**
- 理解每个因子的选股逻辑
- 观察因子在不同滚动年期（1/3/5/7/10年）的表现
- 区分"顺风期"和"逆风期"的全景图

**第四步：组合单因子为多因子模型**
- 像搭积木一样，用53个单因子和31个多因子组件拼搭模型
- 在密切相关因子中选有效性最强的，或构建成综合指标
- 将相互独立的因子组合，让各自优势最大化

**第五步：实盘前做定性复核**
- 模型筛选后速览招股说明书、最近3年年报和最近季报
- 直觉不对的股票要剔除
- 宁可错过上涨股票，也不能坠入模型无法衡量的风险

**第六步：持续迭代**
- 模型不是一成不变的，随视野和经验提升不断改进
- 以10年为单位思考和布局

### 2.2 过程的优先级高于结果

使用"过程与结果二维图"：

| | 好过程 | 坏过程 |
|---|-------|-------|
| **好结果** | 顺理成章的成功 | 好运气（最危险——会让人误以为能力使然） |
| **坏结果** | 坏运气（坚持即可） | 必然的失败 |

- 好过程可能短期遭遇挫折，但终将收获满满
- 坏过程可能一时好运，但终将一无所获
- **决策铁律**：坚持正确的投资过程，穿越好运气与坏运气

## 3. 单因子与多因子模型组合方法论

### 3.1 因子选择的核心逻辑

**组件因子定义**：具有投资价值的单因子或多因子指标，能带来稳定的正或负超额收益。

**相关性处理原则**：
- 低相关因子 → 简单相加构建多因子 → 产生较大超额收益
- 高相关因子 → 选有效性最强的，或构建综合指标 → 避免风险共振

**因子构建要点**：
- 简单多因子并不简单，不要小瞧它们
- 双因子模型有时比多因子模型更有效（因子越多，模型有效性可能越差）
- 组件因子可以用多因子作为组件（嵌套使用）

### 3.2 测试验证的二因子模型范例

| 模型 | 组合因子 | 逻辑 | 20只股票年复合收益率 |
|------|---------|------|-------------------|
| 估值-质量 | 投入资本回报率 + 市销率 | 高创利能力 + 不支付过高价格 | 12.77% |
| GARP | 市盈率 + 净利润增长率 | 合理价格增长策略 | 12.87% |
| 估值-低分歧 | 市净率 + 6个月波动率 | 便宜 + 市场分歧小 | 15.14% |
| 质量-低分歧 | 净资产收益率 + 6个月波动率 | 高创利 + 预期一致 | 16.90% |

### 3.3 五因子模型中的最优表现

最强五因子模型：**市值 + 毛利率 + 投入资本回报率 + 6个月波动率 + 市销率**

- 20只股票年复合收益率：**18.44%**
- 总增长率4038.12%，是整体股票的5.7倍
- 滚动3年期以上战胜整体股票的概率：**100%**
- 3个22年：10万元 → 70.86亿元

### 3.4 构建原则总结

1. 在密切相关因子中，挑选有效性最强的，或者构建综合指标
2. 把相互独立的因子组合在一起，优势最大化
3. 简单多因子并不简单——它们能实现超强回报
4. 因子并非越多越好，可能引发风险共振
5. 选股模型像搭积木——随时可以自由拼搭

## 4. "神奇公式"（格林布拉特）的中国市场适应

### 4.1 原版神奇公式在中国
- 逻辑：高投入资本回报率 + 低企业价值倍数
- 20只股票年复合收益率：10.76%（仅小幅战胜整体股票的9.97%）
- 战胜概率呈先升后降趋势（5年期为拐点）
- **结论：原版在中国市场不尽如人意**

### 4.2 改进版神奇公式
在原有两个要素基础上，增加两个要素：
- **波动率**（未来价格预期一致）
- **毛利率**（上下游地位强势）

改进后结果：
- 20只股票年复合收益率：14.48%
- 总增长率为整体股票的2.6倍
- 战胜整体股票的概率不断提升

### 4.3 中小市值改进版神奇公式（最强版本）
加入第5个要素：
- **市值**（增加小市值倾向）

结果：
- 20只股票年复合收益率：**16.48%**
- 总增长率为整体股票的3.9倍
- 7年期和10年期战胜整体股票概率：**100%**
- 10万元 → 再投资两个22年 → **21.02亿元**
- **结论：可直接用于实战**

### 4.4 神奇公式改进方法论
"秉承其逻辑，在该逻辑基础之上做进一步的研发"
步骤：发现原版不足之处 → 分析中国市场特征 → 逐步添加有效因子 → 验证并迭代

## 5. 6种"低估值陷阱"（价值陷阱识别清单）

价值陷阱定义：买入估值较低的股票，期待价格上涨或估值提升，却发现估值或价格不变甚至下降。

| # | 陷阱类型 | 识别特征 | 典型案例 | 规避策略 |
|---|---------|---------|---------|---------|
| 1 | **景气顶点的周期股** | 宏观经济扩张后期，钢铁/煤炭/水泥/银行/地产等盈利极好，PE极低 | 周期股业绩最好时买入 → 经济收缩 → 净利润暴跌 → 估值飙升 | 林奇警告："在周期股业绩最好的时候买入，只能收获未来的下跌" |
| 2 | **重资产"夕阳"行业** | 行业几乎无增速甚至负增长，股价非常便宜 | 2000年后的彩电行业（长虹）、伯克希尔纺织业 | 巴菲特："一匹会算术的马是了不起的马，但绝不是一个了不起的数学家" |
| 3 | **网络效应行业的中小型公司** | 龙头控上下游资源建壁垒，中小公司仅能自保 | 网络效应行业中的边缘公司 | 即便股价再便宜也不值得购买 |
| 4 | **低水平大幅扩张产能的企业** | 行业产能大规模扩张后严重过剩，价格战刺刀见红 | 多数竞争对手同时扩产 → 行业价格连变动成本都无法收回 | 股价没有最低，只有更低 |
| 5 | **财务报表存在瑕疵** | 净利润是财务报表中最容易被操纵的项目 | 账面净利润高但含金量低 | 检查净利润含金量，警惕操纵隐患 |
| 6 | **"历史底"不是估值底** | 卖方推荐"估值已处历史低位，上涨概率100%" | 历史预期 ≠ 现在预期 | 历史反复证明：没有最低，只有更低 |

**正确低估值股票的标志**（巴菲特）：
"我们喜欢这样的城堡，它有很宽的护城河，河里游满了鲨鱼和鳄鱼，足以抵挡外来的侵入者。"

## 6. Drawdown（回撤）管理与纪律维持

### 6.1 回撤的关键事实（基于22年A股数据）

- **所有回撤最终都会被修复**，只是时间或早或晚、速度或快或慢
- 最优策略的修复时间更快，幅度更小
- 最差策略（第十分位）的回撤幅度、下跌持续期和修复持续期都显著更大
- 滚动投资5年期的规律：**最佳5年期之后往往迎来最差5年期；最差5年期之后往往迎来最佳5年期**

### 6.2 "逆风期"的心态管理

**三个必须认清的事实**：
1. **最优策略也会遭遇"逆风期"**：神奇公式每4年有1年低于市场平均，每6年有连续2年表现不佳，甚至发生过连续3年低于平均水平
2. **战胜市场与赚钱是两码事**：市场下跌10%时，最优策略可能只跌8%，但仍然亏钱
3. **最优分位回撤次数可能多于最差分位**：不是因为差，而是因为弹性足

**行动指南**：
- "逆风期"无法避免——但当它来临时，应庆幸市场给了更大赚钱机会
- 最差5年期之后是未来最好的投资时期——此时应追加资金
- 段永平策略：贵州茅台180元买入，越跌越买，越买越跌，持续数年
- 段永平名言："危机大概5-8年来一次，希望下一次来的时候你记得来这里看一眼，然后擦擦冷汗，把能投进去的钱全投进去。千万别借钱，因为没人知道市场疯狂起来到底有多疯狂。"

**纪律铁律**：
- 守住模型，守住策略，就有东山再起的机会
- 斯波朗迪："如果你赚钱了，请给自己点个赞；如果你亏钱了，但是坚持了自己的投资策略，请给自己点两个赞"
- 只要不溃，严守纪律，即便单只股票或某个时段亏钱了，也不影响长期收益

### 6.3 如奴隶般套用模型

格林布拉特的实证研究：
- 自主管理账户（允许主观判断）：两年总收益率59.4%，低于标普500的62.7%
- 专业管理账户（固定时间买卖）：两年总收益率84.1%，高于标普500的21.4%
- 差异来源：自主账户系统避开了"短期不受欢迎但长期表现好的赢家股"，且倾向持有大量现金

**西蒙斯的建议**："如奴隶般地套用模型。不论你觉得模型是多么的聪明或是愚蠢，你都需要按照模型告诉你的去做。"

## 7. 如何利用回测数据而不陷入过拟合

### 7.1 正确使用回测数据的方法

**看什么**：
- 看22年超长期月度数据结论（1998-2020）
- 看滚动投资1/3/5/7/10年期的全景图
- 看顺风期和逆风期的完整表现
- 看"最差情景预期收益"（均值-1.5倍标准差）

**用什么方法**：
- 筛选股票仅用**简单排序**（从大到小或从小到大）
- 多因子模型仅用**等权重相加**方法
- 不使用微积分、西格玛、线性回归、机器学习、人工智能等复杂方法
- 复杂方程式仅用于业绩评价，不用于选股

**避免过拟合的原则**：
1. 使用22年超长周期月度数据，确保结论稳健
2. 采用简单的排序方法，避免参数过度优化
3. 检验多组滚动年期（不以单一时间窗口为依赖）
4. 观察因子在不同市场环境（牛/熊/震荡）下的表现
5. 确认因子有效性在不同股票池（整体/大盘/小盘）中的一致性

### 7.2 实证数据的正确用途

数据告诉你：
- **什么事情该做**：采纳长期绝对收益高、相对业绩好、战胜基准概率高的策略
- **什么事情不该做**：摒弃长期绝对收益差、战胜基准概率低、长期处于超过20%回撤区间的策略

### 7.3 财务数据的5个固有缺陷

不盲信财务数据——注意以下局限：
1. 财务数据忽略价值创造（不考虑资本成本）
2. 财务数据不能预测未来
3. 财务数据易受会计数据质量影响
4. 财务数据忽略了无形资产
5. 财务数据间容易存在关联性风险

## 8. 量化规则与定性判断的平衡

### 8.1 双轨决策流程

**第一步：量化筛选（70%以上权重）**
- 用书中验证的长效因子构建模型
- 机械执行模型输出结果
- 如奴隶般套用模型筛选股票和负面清单

**第二步：定性复核（辅助判断）**
- 模型筛选后速览招股说明书、最近3年年报和最近季报
- 如果直觉不太对，该股票就会被剔除
- 宁错过，不坠入模型无法衡量的风险

### 8.2 爱因斯坦的提醒
"并非所有能计算清楚的事物都很重要，也并非所有重要的事物都能计算清楚。"

### 8.3 格雷厄姆晚年转变
格雷厄姆最初主张先做定性分析，再通过定量方法决定投资。但40年后他表示，市场已变，大量研究之下很难找到优秀到抵消研究成本的标的。他转而提供纯量化的解决方案。

### 8.4 PROBM模型与"如奴隶般"的启示
- 康奈尔大学MBA学生用PROBM模型发现安然可能存在操纵行为——虽然主观上不相信
- 他们"如奴隶般相信模型结论"——这是他们发现问题的关键
- 投资中，相信模型的能力有时比相信主观感受更重要

## 9. "Besomeone"框架：构建投资能力

### 9.1 核心概念

**Besomeone vs Justdoit**：
- Justdoit 强调"去**做**"
- Besomeone 强调"用大师的思维去**思考**"，"假装"自己就是投资大师，然后再去行动

### 9.2 理论基础

**侯世达《我是个怪圈》**：我们每个人都是一束他人灵魂的碎片，只不过以一种全新的方式组合起来。如果你内部的灵魂碎片是一群伟人，那么你也是一个伟大的人。

**柯维"两次创造"论**：任何事物都是先在头脑中构思（第一次创造），再付诸实践（第二次创造）。先Besomeone，再Justdoit。

### 9.3 具体操作方法

**拿破仑·希尔的方法**：
把想成为的那个人（或那些人）摆在心中最重要的位置。遇到问题，召集心中之人开会：
- 问：林肯先生，如果遇到这种情况，你会怎么做？
- 问：爱迪生先生，你的看法是什么？
- 问：牛顿先生，你如何解决？
然后，听从他们的建议，依照他们的思维去解决问题。

### 9.4 实现闭环的4个条件（刻意练习）

## 10. 关键决策规则与检查清单

### 10.1 选股检查清单

**必须买入的股票特征**：
- [ ] 低估值（市盈率/市净率/企业价值倍数/市销率/市现率在较低分位）
- [ ] 有宽护城河（无形资产/成本优势/转换成本/网络效应/有效规模）
- [ ] 具备竞争优势和行业地位
- [ ] 成长性中等偏上（"太高"与"太低"都不好）
- [ ] 高资本回报率（ROE、ROA、ROIC靠前）
- [ ] 高毛利率（衡量上下游地位的最优指标）
- [ ] 低波动率（价格分歧小）
- [ ] 现金流健康（经营活动现金流与营业利润匹配）
- [ ] 有分红记录（胜过无分红股票约3倍）

**必须规避的股票特征（负面清单）**：
- [ ] 市盈率为负值（作为反向指标——其越涨越说明牛市快结束）
- [ ] 企业价值倍数为负值
- [ ] 高市净率（尤其第七分位"伪成长股"）
- [ ] 高波动率 + 低动量（最差组合）
- [ ] 触发危险信号（现金流/利润比率异常、应收账款周转天数过长等）
- [ ] 属于6种价值陷阱之一
- [ ] 第十分位的任何估值因子
- [ ] PROBM模型标记为潜在操纵者

### 10.2 交易决策规则

**买入时机**：
- 遭遇"逆风期"且策略未变 → 加仓
- 最差5年期结束时 → 大举加仓
- 危机发生时（约5-8年一次）→ 投入所有可投资金
- **铁律**：绝不借钱（没人知道市场会疯狂到什么程度）

**持有/卖出**：
- 以10年为单位布局
- 短期（1年）→ 60%收益来自估值变化，不必在意
- 长期（5年）→ 80%收益来自购买成本和企业质量
- 模型触发危险信号 → 剔除
- 出现更好的替代标的 → 换仓

### 10.3 风险管理清单

真正的风险来源（非贝塔/标准差）：
- [ ] 盈利能力不足的企业
- [ ] 购买价格过高
- [ ] 极端乐观情绪
- [ ] 故意忽略风险或认为风险不存在

风险控制措施：
- 先损失最小化，再追求收益最大化（"先为不可胜，以待敌之可胜"）
- 一定在能力圈内投资
- 任何时候都留安全边际
- 极端乐观时要警惕（"在别人热切买入时卖出"）
- 永恒记住：风险永远会以想不到的方式出现

## 11. 因子周期性变化与市场风格切换

### 11.1 价值股与成长股的"跷跷板效应"

基于实证数据的关键发现：
- 价值股在22年长周期内跑赢成长股
- 但在1-3年短周期内，成长股可能屡次优于价值股
- 大盘股票池中的跷跷板效应比整体股票池更加明显
- 美国市场风格切换非常慢（18年9个月仅1次切换），中国市场切换较快（4次）

**对投资者的启发**：
- 不要因为成长股短期大涨就放弃价值策略
- 也不要在价值股好时忘记成长股的存在（"这次不一样了"是危险的错觉）
- 引用贺拉斯："现在已然衰朽者，将来可能重放异彩；现在备受青睐者，将来可能黯然失色"

### 11.2 因子的生命周期

因子有效性存在周期性变化，经历5个阶段：

1. **不被关注，但有效性极好** → 最佳投资时机
2. **逐步引起关注** → 资金开始流入
3. **被大量使用，实际效果开始下降**
4. **完全拥挤，效果还不如"扔飞镖"选股**
5. **被弃用** → 有效性逐步修复，直到引起下一轮关注

企业价值倍数在中国市场的典型案例：
- 2013年前：非常有效的单因子，显著筛选出优异股票
- 2014年后：逐渐失效，选出股票的业绩几乎与被动投资整体股票相当

**行动指南**：
- 当因子表现最好且人人都在谈论时 → 谨慎（可能已处于第3-4阶段）
- 当因子表现很差且被所有人遗忘时 → 关注（可能处于第5阶段，即将进入新一轮有效周期）

### 11.3 如何判断身在周期何处

虽然不能精确预测，但可以做到：
- 知道周期永远存在（经济周期、信贷周期、存货周期、市场情绪周期）
- 知道趋势迟早会反转
- 知道在极端位置应采取相应行动

**周期信号**（基于实证）：
- 市盈率为负值股票的超额收益转正 → 牛市可能结束
- 市现率为负值股票涨势超越整体股票 → 牛市盛宴可能终结
- 第十分位股票在牛市末期大幅跑赢 → 最后疯狂信号
- 企业价值倍数最高分组在牛市末期表现最好 → 牛市即将结束

**邓普顿规则**："在别人绝望抛售时买进，在别人热切买入时卖出——这样需要最大的决心，同时也会让我们获取最大的收益。"

### 11.4 耐心是策略的基石

- 从1年角度看：60%收益来自估值变化（价格波动）
- 从5年角度看：80%收益来自购买成本和企业质量
- 短期的噪音中，"做点儿什么"的冲动常常是错误的
- 老子："无为才能有为"——很多人为了显得"有为"而错失了真正的好风
- 巴菲特："偶然一时做出的成功投资，却需要长期无所作为静静去等待"
