> **书籍**: 《基本面量化投资策略》
> **作者**: 董鹏飞
> **出版社**: 中信出版集团, 2022年
> **分析日期**: 2026-05-14

## 1. 句式模式（Sentence Patterns）

### 1.1 核心句式结构：数据前置 — 结论收束

董鹏飞最典型的句式是**「先摆数据，再下结论」**的实证主义三段论：

```
[数据来源/回测区间] + [分组方式/指标定义] + [数值结果] + [因此/由此可见] + [结论/投资建议]
```

**典型例句：**
- "从1998年12月31日至2020年8月31日共21年8个月的数据显示，超级大盘股模拟组合的年复合收益率为8.34%，大盘股为8.44%，整体股票为10.5%，超级小盘股为13.9%。"
- "从图3-1和图3-2可以看出，随着股票市盈率从低到高变化，各个分位股票模拟组合的年复合收益率呈现总体递减的趋势。这符合金融学的基本逻辑：市盈率越高，估值越高，未来收益空间越有限；市盈率越低，估值越低，未来收益空间越大。"

### 1.2 对比对照式

作者频繁使用中美对比、高低分组对比、长短周期对比来强化说服力：

- "对照詹姆斯·奥肖内西对美国股票市场从1964年1月1日至2009年12月31日所做的统计，我们可以得出如下结论：第一，美国表现最优的是市盈率最低分组（第一分位），中国表现最优的是第三分位；第二，无论是中国还是美国，表现最差的都是市盈率最高分组（第十分位）；第三，中国各个十分位年复合收益率差额的最大值为5.85%，远远低于美国股票市场的10.72%。"

### 1.3 设问引导式

在章节开头或关键转折处，用设问句吸引读者：

- "既然有了会计利润，为什么还要统计现金流？"
- "既然有了这么简单明确的答案，为什么还要创造复杂缥缈的'伪答案'呢？"
- "下一个问题是，能力可以提升吗？"

### 1.4 排比递进式

用于总结或强化观点：

- "要么不战，等待机会；要么战，战则必胜"
- "对的越对，错的越错"
- "一琴通，百琴通，一琴精，百琴精"

## 2. 修辞手法（Rhetorical Devices）

### 2.1 数据驱动的实证说服（最核心手法）

全书通过近300幅图表、53个单因子指标和31个多因子指标构建说服力。作者几乎从不做无数据支撑的价值判断。每当提出一个观点，必定紧跟实证数据。

**典型模式**：断言 → 引用文献来源 → 展示回测数据 → 给出投资启示

### 2.2 历史故事类比

在抽象数据之间植入生动的历史/文学典故，缓解阅读疲劳：

- **刘备故事**：以刘备46岁请诸葛亮、遭遇最大回撤"仅数十骑逃走"为例，说明正确策略也可能遭遇短期巨大挫折
- **孔子学琴**：以孔子学《文王操》为例，阐述Besomeone理念
- **孙叔敖封地**：以孙叔敖要求儿子选择贫瘠之地为例，说明"着眼长久"的价值
- **守株待兔**：以农夫守株为反例，说明过程与结果的关系
- **牛顿南海泡沫**："我可以计算天体的运行轨道，却无法计算人性的疯狂"

### 2.3 大师名言背书

每一章开头引一位大师名言，正文中高频引用大师观点来佐证数据结论：

- 巴菲特、格雷厄姆、芒格、林奇、邓普顿、马克斯、卡拉曼、段永平、格林布拉特等
- 引用方式通常是：引用名言 + "这与我/数据的结论一致"

### 2.4 比喻与类比

- **军事类比**："冷兵器军队遭遇现代化部队"（散户 vs 量化）
- **医学类比**："钝刀子割肉"（描述长期回撤的痛苦）
- **自然类比**："树木不会长到天上去"
- **体育类比**：泰德·威廉斯的"甜蜜击球区"、放风筝等风来
- **风/水/路径比喻**："逆风期"、"顺风期"、"长长的坡，湿湿的雪"

### 2.5 正反对比论证

每个因子指标都同时分析"优秀分组"和"差劣分组"，形成鲜明的正反对照：

- "低市盈率分组值得关注" vs "尽量规避高市盈率分组"
- "低市净率分组值得关注" vs "高市净率分组表现不佳"
- "低市现率分组值得期待" vs "高市现率分组表现最差"

### 2.6 警示性语言

对风险提示采用带有情感色彩的警示句式：

- "千万不要轻易做空市净率较高的股票"
- "我们离这些股票越远越好"
- "这真令人心焦，我实在想不出投资该策略的理由"
- "亏起钱来'哗哗的'，不能轻易言底"

## 3. 词汇特征（Vocabulary Traits）

### 3.1 高频定量术语

- **回测参数类**：年复合收益率、十分位法/五分位法、整体股票池/大盘股票池、滚动N年期、模拟组合、夏普比率、索提诺比率、最大回撤、标准差
- **分组用语**：第一分位/第十分位、前N分位/后N分位、低估值分组/高估值分组
- **时间用语**：过往22年、21年8个月、21.75年、牛市/熊市、顺风期/逆风期

### 3.2 独特的概念语言

作者创造或引用了一系列形象化的概念来建立"专属词汇体系"：

| 概念 | 含义 |
|------|------|
| 低估值陷阱 | 看似便宜实则昂贵的股票 |
| 伪成长股 | 有过高成长历史但增速下滑、估值仍偏高的股票 |
| 跷跷板效应 | 价值股与成长股之间的轮动现象 |
| 危险信号 | 可用于识别差股票的财务指标 |
| 因子有效性轮回 | 因子从不被关注到被滥用再到有效性恢复的周期 |
| Besomeone | 以大师思维去思考，先"成为"再行动 |

### 3.3 语言风格定位

**整体定位：学术严谨 + 实务可读**

- **学术面**：大量使用金融学术语（贝塔、阿尔法、标准差、夏普比率、索提诺比率），引用学术文献（法玛-弗兰奇三因子模型、巴苏市盈率研究等）
- **通俗面**：避免过度数学化，用"10万元投资最终变成X万元"代替复杂的复利计算；用"旧时王谢堂前燕，飞入寻常百姓家"等古诗词点缀
- **口语化穿插**："擦擦冷汗，把能投进去的钱全投进去"、"把书都读到狗肚子里了"、"钝刀子割肉"

### 3.4 措辞风格特征

- **倾向使用"我们"**：拉近与读者距离，形成"共同探索"的陪伴感
- **适度使用"我"**：在表达个人观点或经历时使用，如"我认为"、"我一般会结合"、"在我投资中"
- **大量使用"或许"、"可能"、"通常"**：保持学术审慎
- **力度较强的断言**："千万不要"、"一定要"、"务必"——在给出明确建议时完全不留模棱两可的空间

## 4. 结构模式（Structural Patterns）

### 4.1 全书整体结构

```
第1章 投资哲学（第一目的/第一性原理）          ← 形而上的"道"
第2章 规模效应（市值因子）                     ← 最基础的因子
第3-8章 六大估值指标（PE/PB/EV/PS/PCF/股息率）  ← 核心估值体系
第9章 成长性指标                              ← 增长维度
第10-11章 资本回报率与利润率                   ← 质量维度
第12章 危险信号                                ← 排雷维度
第13章 21个财务指标（5大类）                   ← 扩展财务维度
第14章 动量/反转/波动率                        ← 价格行为维度
第15章 多因子模型 + 神奇公式改进                ← 综合组装
结语  Besomeone 投资哲学升华                    ← 回到"道"
附录 财务报表分析质量                           ← 实用工具
```

### 4.2 单章内部结构

每个章节遵循几乎固定的五段式结构：

1. **大师引言**：以投资大师名言开篇
2. **指标溯源**：追溯该指标的学术起源（"XX年，XX发表了《》一文"）
3. **中国实证**：展示中国A股回测数据 → 十分位柱状图 + 统计表格
4. **优劣分析**：低分组（第一分位）表现 × 高分组（第十分位）表现 → 正反对比
5. **投资者启迪**：以"对投资者的启迪"为标题，列出N条可操作的行动指南

### 4.3 标准论证流程

```
理论假设 → 学术溯源 → 中国数据验证 → 横向对比（中美/高低） → 现象解释 → 行动建议
```

### 4.4 独特的小结方式

每个策略章节末尾必有"对投资者的启迪"，以编号列表（1,2,3...）形式给出，语言从"描述性"切换为"指令性"。

## 5. 引用习惯（Citation Habits）

### 5.1 最常被引用的大师

按出现频率排序：

1. **沃伦·巴菲特** ── 绝对核心引用对象，每章至少出现1-2次
2. **本杰明·格雷厄姆** ── 基础理论提供者（安全边际、市场先生）
3. **查理·芒格** ── 格栅理论、多元思维模型、误判心理学
4. **彼得·林奇** ── 选股实务、"从生活中发现10倍股"
5. **霍华德·马克斯** ── 周期理论、风险认知
6. **乔尔·格林布拉特** ── 神奇公式、企业价值倍数
7. **约翰·邓普顿** ── 逆向投资、"税后总回报最大化"
8. **塞思·卡拉曼** ── 安全边际、相对业绩批判
9. **段永平** ── "买股票就是买公司"、"危机投资"
10. **约翰·博格** ── 指数投资、金融行业批判

### 5.2 学术文献引用

- **法玛 & 弗兰奇 (1992/1993)**：三因子模型
- **巴苏 (1977)**：市盈率研究
- **班次 (1981)**：规模效应
- **布鲁姆 & 胡斯克 (1973)**：规模效应发现

### 5.3 引用方式特征

- 引用大师名言时，通常放在**章节开篇**作为题记
- 在正文中引用时，采用**"XX认为/曾说/曾告诫"**句式
- 中国大师引用较少，主要是段永平、张磊、李录（少量）
- 喜欢引用**非金融类作品**：孙子兵法、老子道德经、孔子学琴等

## 6. 叙事风格（Narrative Style）

### 6.1 三重身份的切换

作者在书中扮演三种身份，在不同段落间切换：

| 身份 | 语气 | 用法 |
|------|------|------|
| **学者/研究员** | 严谨、客观、数据化 | 展示回测结果时 |
| **私募从业者** | 经验谈、行内话 | 讨论实盘经验时 |
| **投资教练/导师** | 鼓励性、引导性 | "对投资者的启迪"部分、结语 |

### 6.2 个人经历的有限嵌入

作者有节制地使用个人经历，包括：
- 在FOF工作期间的观察
- 自学编程的经历
- 写作本书的历程
- 个人以10年为单位的投资布局

这些个人经历穿插在学术论述之间，起到"软化"学术文本的作用。

### 6.3 叙事节奏控制

- **密集数据段**（3-5段纯数字）→ **故事/典故段**（1段历史故事）→ **结论建议段**（编号列表）
- 这种"紧-松-紧"的节奏交替防止读者阅读疲劳
- 每个"对投资者的启迪"都是一个"呼吸窗口"

### 6.4 情感色彩

整体克制中的适度流露：
- 在描述第十分位（差股票）时：**"这真令人心焦"、"恐怖"**
- 在描述好策略时：**"令人兴奋"、"超出预期"**
- 情感调动集中在"逆风期"的讨论中，引导读者"坚持住"
- **总体基调**：理性乐观主义 — 数据说好就是好，数据说不好就规避

## 7. 确定性语言（Certainty Language）

### 7.1 确定性光谱

作者使用了一套明确的分级确定性语言体系：

| 确定性等级 | 典型用语 | 使用场景 |
|-----------|---------|---------|
| **绝对确定** | "一定要"、"千万不要"、"务必"、"是必须规避的" | 投资行动建议 |
| **高度确信** | "能够有效筛选"、"呈现严格递减"、"值得关注" | 实证结论陈述 |
| **中度确信** | "或许"、"可能"、"一般来讲"、"通常" | 解释原因时 |
| **审慎保留** | "可能的原因是"、"尚需进一步验证" | 理论推测时 |
| **承认局限** | "模型不是死的"、"要警惕模型不能衡量的风险" | 量化局限讨论 |

### 7.2 数据确定 + 解释不确定

- 对**数据本身**：高度自信（有回测结果支撑）
- 对**数据的原因**：适度保留（"可能的原因是" "这可能是因为"）
- 对**未来的表现**：基于历史概率，但留有余地

### 7.3 经典的"但是"转折

这是作者最重要的修辞策略之一：

- "即便是最优策略，也不是任何时候都最优。在特定时间内，也会面临回撤。"
- "虽然价值股在长周期内跑赢成长股，但在1～3年的短周期内，成长股的表现可能屡次优于价值股。"
- "简单多因子并不简单，不要小瞧它们。"

这个"但是"结构既表达了实证结论的力度，又避免了过度承诺。

### 7.4 自我反思与边界认知

作者在某些地方主动展示自我怀疑和局限认知：
- "在写本书之前，很多概念混淆不清。为了写作本书，我进行了大量阅读和研究，逐一攻克了这些难题。"
- 关于会计学："有时候，我觉得会计是一门魔术。"
- "爱因斯坦认为，并非所有能计算清楚的事物都很重要，也并非所有重要的事物都能计算清楚。"

## 8. 综合表达DNA特征总结

### 8.1 核心DNA：实证主义叙事（Evidence-First Narrative）

董鹏飞的表达DNA最核心的特征是：**所有观点都必须经由中国A股数据验证**。他不是先有观点再找数据支持，而是先看数据再形成观点。这使他的写作风格区别于两类常见投资写作者：
- 区别于**哲学式**（如霍华德·马克斯）—— 不依赖大量数据回测
- 区别于**经验式**（如彼得·林奇）—— 不依赖个人投资故事
- 区别于**纯学术式**（如法玛）—— 不以数学模型本身为目的

他的独特定位是：**用数据讲清楚一个投资策略在中国到底管不管用**。

### 8.2 二阶DNA：翻译者姿态

作者扮演"学术界研究成果 → 中国A股市场 → 普通投资者"的三层翻译者角色：
1. 把英文学术论文翻译为中文投资语言
2. 把美国市场的结论翻译为中国A股的实证检验
3. 把复杂的回测结果翻译为"10万元变成X万元"的可感知效果

### 8.3 三阶DNA：置信区间写作法

作者在确定性和不确定性之间建立了明确的"置信区间"：
- 数据结论 = 高置信区间（95%以上确信）
- 原因解释 = 中置信区间（60%-80%确信）
- 未来预测 = 概率表述（"滚动5年期战胜概率为X%"）

他很少做出无概率陈述的预测，凡是预测必定附带历史概率。

### 8.4 四阶DNA：系统构建者思维

全书本质上是一个**积木式系统**的教学手册：
- 第1章：为什么需要系统（道）
- 第2-14章：单块积木（单因子）
- 第15章：如何搭积木（多因子）
- 结语：成为搭积木的人（Besomone）

这种由零件到整体的构建思维渗透在每一章的写作方式中，使全书既是"说明书"又是"工具箱"。

### 8.5 五阶DNA：温和拯救者叙事

作者隐含的叙事姿态是"拯救散户于非理性投资"：
- 开篇定义"投资的第一目的"——纠正被金融机构误导的目标
- 中间展示回测数据——纠正凭感觉投资的错误习惯
- 结语"组装灵魂碎片"——提升读者到"投资大师"的层次
- 整体叙事弧线：困惑 → 数据揭示真相 → 建立系统 → 成为大师
